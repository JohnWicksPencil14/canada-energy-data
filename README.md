# Canada Energy Data

Province-level Canadian energy data maintained by [reachdata.ca](https://reachdata.ca) — Canada's energy intelligence platform for AI infrastructure planning.

## Dataset

**File:** `canada-energy-data.csv`

Covers all 10 Canadian provinces with the following fields:

| Field | Description |
|-------|-------------|
| `province` | Province name |
| `abbr` | Province abbreviation |
| `primary_source` | Primary energy generation source |
| `installed_capacity_gw` | Total installed generation capacity (GW) |
| `clean_energy_pct` | Percentage of electricity from clean sources |
| `avg_price_cents_per_kwh` | Average electricity price (cents/kWh) |
| `grid_operator` | Provincial grid operator |

## Why This Data Matters

Canada is emerging as the world's most strategic location for AI data center infrastructure:

- **Abundant clean energy** — Quebec and BC run on 98-99% clean hydro
- **Lowest electricity costs** in North America in key provinces
- **Cold climate** — natural cooling reduces data center operating costs
- **Political stability** — reliable regulatory environment for long-term investment
